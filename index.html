HTML
  1. Building HTML
    • File (name.html)
    • Language declaration (html lang="pl-PL")
    • Basic Tags (html, head, body)
    • Data in tags in body (tagName attributeName="attributeValue" Data tagName)
    • Special chars added by reference equivalent (&amp;)
    • Comments ()
  2. HEAD tag
    • Set characters (meta charset="utf-8")
    • Author (meta name="author" content="")
    • Description (meta name="description" content="Description under tite in Google Search")
    • Add external styles (link rel="stylesheet" href="filePath/fileName.css")
    • Add external scripts (script src="filePath/fileName.js" type="text/javascript" defer /script)
    • Icon before title (link rel="shortcut icon" href="faviconName.ico" type="image/x-icon")
    • Title tag
  3. Body tag
    1. Text
      • Paragraph (p)
      • Headings, Sub-headings (h1-h6)
      • Wrappers (div, span)
      • Line break (br)
      • Horizontal rule (hr)
      • Strong
      • Whitespace is only one
    2. Link
      • a href="linkPath"
      • Link desription (title="")
      • Image can be a link
      • Link to document by id in href
      • Absolute, Relative URL.
      • Email link (a href="mailto:mailAddress")
      • Good link practise:
        • Link name: short, descriptive.
        • Use relative links.
        • Downloading links name: Download pdf (10MB), Watch wideo (strean opens in tab), Play the game (requires flash)
        • Downloading links uses download attribute to set default fileName.
    3. Advanced Text
      • Description list (dl-list, dt-term, dd-definition)
      • Lists (ol, ul, li)
      • Block quotations (blockquote)
      • Line quotations (q)
      • Cite (no use)
      • Abbreviations (abbr title="fullAbbreviationName)"
      • Address
      • Superscript, Subscript
      • Code (code, pre, var, kbd, samp)
      • Time (time datetime="2020-01-31T12:00"):
    4. Images
      • img src="imagePath"
      • No image display description (alt="")
      • If image is decoration alt should be blank
      • Alt should be descriptive
      • Img can be a link
      • Image title
      • Image with description (figure)
      • Title inside figure (figcaption)
      • Responsive (srcet="imgPath maxWidthw, imgPath maxWidthw")
      • Responsive (sizes="(max-width: widthpx) maxWidthpx, maxWidthpx")
      • Picture tag (picture, source)
    5. Vector Graphics
      • Vectop graphics tag (svg width="" height="")
      • Circle (circle cx="" cy="" r="" fill=""
      • Rectangle (rect width="" height="" fill=""
      • We can add svg in img tag
    6. Audio/Video
      • video src=""
      • Multiple video formats (source src="" type="video/mp4")
      • Preset settings (controls, autoplay, loop, muted, poster="imagePath", preload="none/auto/metadata"
      • audio src=""
      • Multiple audio formats (source src="" type="video/mp4")
      • track kind="subtitles" src="subtitlesPath.vtt" srclang="" label="Polish"
      • Subtitles.vtt: WebVtt, number, time of track --> time of displaying subtitles, subtitles
    7. Tables
      • table
      • Table cell (td)
      • Table row (tr)
      • Table header (th), scope="col/row" is for screen readers
      • Connecting rows (rowspan="nrOfRows")
      • Connecting columns (colspan="nrOfColumns")
      • Styling tables (colgroup, col), u can use in col tag span="nrOfColumns" attribute
      • Table title (caption)
      • Table structure (thead, tbody, tfoot), table styling should be inside thead
      • Assotiation between headers and cells (th id="idName", td headers="idName")
    8. Embending
      • iframe src="webPageURL"
      • Attributes (allowfullscreen, frameborder="0/1" sandbox)
      • Safety: use when necessary, https, sandbox mode
      • Embed, object elemets
    9. Document Structure
      • Header
      • Nav
      • Main (main, article, section, div)
      • Aside
      • Footer
  4. Debugging HTML
    • Syntax - wrong, no close, wront letter, attribute.
    • Logic - syntax is ok but you didn't want that HTML structure.
    • Validation - checking is everything ok with syntax, logic, semantic.
    CSS
  1. Building CSS
    • File (name.css)
    • Internal CSS (style Style Rules /style)
    • External CSS ((link rel="stylesheet" href="filePath/fileName.css"))
    • Syntax (elementSelector {property: value;})
    • Comments (/* Text */))
    • CSS is cascading (styles after overwrites before)
    • Rules (@import 'style.css')
    • Functions (calc(90% - 30px)
    • Formatting CSS
    • Comments
    • Logical sections/groups
    • Avoid overly-specific selectors
    • Break CSS into multiple files
    • Variables
    • BEM
    • Frameworks, Preprocessors
  2. Selectors
    • Universal (*)
    • Class (.className)
    • Id (#idName)
    • HTML (tagName tagName)
    • Attribute (a[attr], a[attr="value"], attr~/|/^/$/*="value")
    • Pseudo class ()
    • Pseudo elements (::after/before/backdrop/cur/cur-region/first-letter/first-line/file-selector-button/grammar-error/marker/part()/placeholder/selection/slotted()/spelling-error/target-text)
    • Combinators (elem >,~,+,|| elem)
    • Several elements (element, element)
    • State (elementName:link/visited/hover/active)
    • We can connect selectors (elementSelector.className/idName tagName)
    • Specificity (!important/styleAttribute, id, class/attribute/pseudoClass, elements/pseudoElement)
    • Shorthands (background)
  3. CSS Rules
    • Controlling inheritance (inherit, initial, unset)
    • Z-index
    • Box model [box-sizing:content/border-box](content, padding, botder, margin)
    • Display (block, inline, inline-block, flex, grid)
    • Background (background-color/image[url()/linear/radial-gradient()]/repeat[no-repeat, repeat/-x/-y]/size[cover/contain/size]/position/attachment[scroll/fixed/local])
    • Border (border-top/bottom/left/right/width/style[dashed/dotted/double/groove/inset/outset/ridge/solid]/color/radius)
    • Writing-mode (horizontal-tb, vertical-lr/rl)
      • Width (inline-size)
      • Height (block-size)
      • Padding, Margin, Border (padding/margin/border-inline/block-start/end)
      • Top, Bottom, Right, Left (inline, block-start/end)
    • Overflow/-x/-y (visible, hidden, auto, scroll)
    • Overflow-wrap, Word-break
    • Values (%, numbers, color[keyword, rgb/a, hsl/a], images[url, gradient], position, string, functions)
    • Units (cm, mm, Q, in, pc, pt, px, em, ex, ch, rem, lh, vw, vh, vmin, vmax)
  4. Item size
    • Margin, Padding % are calculated from content box size.
    • Min/Max sizes (min/max-height/width
    • Viewport
    • Images (object-fit[cover, contain, fill])
  5. Tables
    • Table-layout (auto, fixed)
    • Border-collapse (separate, collapse)
  6. Text
    1. Plain
      • Color[keyword, rgb/a, hsl/a]
      • Font (fonr-family[sans-serif, serif, cursive, fantasy, mono-space])
      • Font-size
      • Font-style (normal, italic, oblique)
      • Font-weight (normal, bold, lighter, bolder, 100-900)
      • Text-transform (none, uppercase, lowercase, capitalize, full-width)
      • Text-decoration-line(none, underline, overline, line-through)/color/style(solid, double, dashed, dotted, wavy)/thickness(auto, form-font)
      • Text-shadow (horizontal, vertical, blur radius, color)
      • Layout (text-align[left, right, center, justify])
      • Line-height
      • Letter/Word-spacing
      • We can add our own font (@font-face {font-family: "fontName";, src: url("fortPath");})
    2. List
      • List-style-type (disc, circle, square, decimal, roman, greek, latin, "\1f44D", other)
      • List-style-position (inside, outside)
      • List-style-image (url())
      • Counting - start="4" attr
      • Reverse - reversed attr
      • Value - value="5" attr
    3. Link
      • Default properties
      • Link is styled in every state (:visited/hover/focus/active)
  7. Layout
    1. Normal Flow
      • Default element layout (block, inline, inline-block)
      • Block elmeents (address, article, aside, blockquote, details, dialog, dd, div, dl, dt, fieldset, figcaption, figure, footer, form, h1-h6, header, hgroup, hr, li, main, nav, ol, ul, p, pre, section, table)
      • Inline elements (a, abbr, acronym, audio, br, button, canvas, cite, code, data, datalist, del, em, i, iframe, img, input, label, mark, map, object, picture, q, script, select, span, strong, sub, sup, svg, template, textarea, time, u, var, video, wbr)
    2. Flexbox - 1 dimension
      • Display (flex, inline-flex)
      • Flex-direction (row, column, row/colum-reverse)
      • Flex-wrap (nowrap, wrap, wrap-reverse)
      • Flex-flow (flex-direction + flex-wrap)
      • Element size (flex)
      • Align-items/self (stretch, center, start, end)
      • Justify-content (start, center, space-between/around/evenly)
      • Order
    3. Grid - 2 dimensions
      • Display (grid, inline-grid)
      • Grid-template-columns (fr, repeat(number[3], what[1fr]))
      • Grid/column/row-gap, gap
      • Grid-auto-columns/rows (auto/ fr/ min-content/ minmax(min, max), )
      • Grid-column/row-start/end
      • Grid-template-areas ("a a a/b c c/ b c c"), in item grid-area: name[a/b/c]
    4. Float
      • Float (none, left, right)
      • Modifies computed display value
      • Clearing float:
        • ClearElement (clear: left/right)
        • ::after (content: ""; clear: both; display: block;)
        • Overflow (auto)
        • Display (flow-root)
    5. Position (static, relative, absolute, sticky, scroll, fixed)
    6. Multiple column layout
      • Column-count
      • Column-width
      • Column-gap
      • Column-rule (size style color)
      • Column-span (none, all)
      • Break-inside (auto, avoid, avoid-page/column/region)
